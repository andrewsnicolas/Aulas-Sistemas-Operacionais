# 🐧 Sistemas Operacionais — Shell Script & Linux

Bem-vindo ao meu repositório da disciplina de Sistemas Operacionais.
Aqui estão armazenados exercícios, scripts, práticas de terminal e desafios utilizando Shell Script e ferramentas do Linux propostas durante as aulas da faculdade.

O foco deste repositório é desenvolver conhecimentos sobre funcionamento de sistemas operacionais, automação de tarefas, gerenciamento de processos e utilização do terminal Linux.

## 🧠 Conteúdos Estudados

Atualmente, os exercícios envolvem temas como:

* Comandos básicos do Linux
* Navegação entre diretórios
* Manipulação de arquivos e pastas
* Permissões de acesso
* Shell Script (bash)
* Variáveis e entrada de dados
* Estruturas condicionais no shell
* Loops (for, while)
* Processos e monitoramento
## 🛠️ Tecnologias Utilizadas
- Linux 
- Bash Shell
- Ubuntu
- Terminal Linux
- VS Code / Nano / Vim

## ▶️ Como Executar os Scripts

Dar permissão de execução:
```shell
chmod +x script.sh
```
Executar:
```shell
./script.sh
```
Ou diretamente pelo bash:
```bash
bash script.sh
```
